# Чеклист тестирования на Beget (MVP)

## Подготовка

- [ ] Создать VPS на Beget: **2 ядра / 2 ГБ / 30 ГБ, Ubuntu 22.04** (твой сервер)
- [ ] Создать VPS на Beget: **1 ядро / 2 ГБ / 15 ГБ, Ubuntu 22.04** (тестовый сервер пользователя)
- [ ] Создать Telegram бота у @BotFather → получить токен
- [ ] Создать второго Telegram бота (для пользователя) → токен
- [ ] Заполнить `.env` на твоём сервере:
  ```
  BOT_TOKEN=<токен твоего бота>
  DATABASE_URL=postgresql+asyncpg://user:password@localhost:5432/jarvisfather
  REDIS_URL=redis://localhost:6379/0
  ENCRYPTION_KEY=<сгенерировать: python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())">
  ```
- [ ] Установить Docker на твоём сервере
- [ ] Клонировать репозиторий на твой сервер
- [ ] `make up && make migrate`

Проверяем на VPS пользователя (через SSH):

```bash

node --version          # v20.x

# Агенты на месте
ls ~/.claude/agents/    # ecommerce-specialist.md, marketing-content.md, presentations.md

# Бот запущен
systemctl --user status jarvis_bot  # active (running)

# База данных создана
ls ~/jarvis_bot/        # bot.py, .env, history.db
```
